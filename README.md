# Meet Transcript Clipper

Google Meetの字幕を自動的に記録・保存するChrome拡張機能です。

## 機能

- Google Meetの字幕テキストと発言者名を自動で記録
- セッション（会議）ごとに字幕履歴を管理
- Markdown / JSON / テキスト形式でのエクスポート
- 全データはブラウザ内にのみ保存（外部送信なし）

## 利用ガイドライン

この拡張機能は、会議の字幕テキストを記録します。ご利用にあたっては、以下のガイドラインを守ってください。

### 参加者への事前告知

**会議の参加者に、字幕を記録していることを事前にお伝えください。**

字幕の記録は参加者の発言内容と氏名を含むため、他の参加者が記録されていることを認識していない状態での利用は、信頼関係やプライバシーの観点から問題となる可能性があります。

会議の冒頭で「議事録作成のために字幕を記録しています」と一言伝えることを推奨します。

### データの取り扱い

- 記録されたデータはお使いのブラウザのローカルストレージにのみ保存されます
- 外部サーバーへのデータ送信は一切行いません
- エクスポートしたファイルの管理・共有は利用者の責任で行ってください
- 不要になったセッションデータはポップアップ画面から削除できます

### 適切な利用例

- 自分の振り返りのための議事録作成
- チームで合意の上での会議記録
- アクセシビリティ支援（聴覚に不安のある方の会議内容の確認）

### 不適切な利用例

- 参加者に無断での発言記録
- 記録データの無断共有・公開
- 個人の発言を監視目的で使用すること
