目的:
Loop 263 で追加された録音状態リングライトは、`TranscriptView` の状態に連動して表示/非表示になります。ただし `TranscriptView` が unmount される場合、既存ライブ字幕 window は cleanup で非表示へ戻している一方、リングライト window には同等の cleanup がありません。録音状態透明性を担う表示が画面遷移や unmount 後に残るリスクを下げるため、unmount 時にリングライト window を非表示へ戻してください。
