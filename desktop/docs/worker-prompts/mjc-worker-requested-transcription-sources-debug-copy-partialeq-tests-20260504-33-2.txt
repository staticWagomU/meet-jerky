### T1: `requested_transcription_sources_debug_output_contains_struct_name_and_both_field_names`
- 目的: `Debug` 出力に「型名 `RequestedTranscriptionSources`」+「両 snake_case field 名」+「bool 値 `true` / `false`」が含まれる契約を CI 固定。
- 期待: 4 種類の case (両 true / 両 false / mic-only / system-only) で以下を assert:
  - 型名 `"RequestedTranscriptionSources"`
  - field 名 `"use_mic"` / `"use_system"`
  - bool 値 `"true"` / `"false"` が両方の Debug 出力に含まれる (`true / false` の文字列形 - Rust の Debug 実装の標準)
- 実装ヒント:
  ```rust
  #[test]
  fn requested_transcription_sources_debug_output_contains_struct_name_and_both_field_names() {
      // 両 true case
      let both = RequestedTranscriptionSources {
          use_mic: true,
          use_system: true,
      };
      let s_both = format!("{:?}", both);
      assert!(s_both.contains("RequestedTranscriptionSources"), "Debug should contain type name: {}", s_both);
      assert!(s_both.contains("use_mic"), "Debug should contain field 'use_mic': {}", s_both);
      assert!(s_both.contains("use_system"), "Debug should contain field 'use_system': {}", s_both);
      assert!(s_both.contains("true"), "both-true Debug should contain 'true': {}", s_both);

      // 両 false case
      let neither = RequestedTranscriptionSources {
          use_mic: false,
          use_system: false,
      };
      let s_neither = format!("{:?}", neither);
      assert!(s_neither.contains("RequestedTranscriptionSources"), "Debug should contain type name: {}", s_neither);
      assert!(s_neither.contains("use_mic"), "Debug should contain field 'use_mic': {}", s_neither);
      assert!(s_neither.contains("use_system"), "Debug should contain field 'use_system': {}", s_neither);
      assert!(s_neither.contains("false"), "both-false Debug should contain 'false': {}", s_neither);

      // 混合 case (mic-only) で true と false が両方とも出ることを確認
      let mic_only = RequestedTranscriptionSources {
          use_mic: true,
          use_system: false,
      };
      let s_mic = format!("{:?}", mic_only);
      assert!(s_mic.contains("true"), "mic-only Debug should contain 'true': {}", s_mic);
      assert!(s_mic.contains("false"), "mic-only Debug should contain 'false': {}", s_mic);
  }
  ```

### T2: `requested_transcription_sources_copy_semantics_allow_use_after_move`
- 目的: `#[derive(Copy)]` の **Copy semantics** 契約 = let copied = original 後も original が使える契約 = move されない = mjc-main-32 Loop 2 で TranscriptionSource enum で確立した Copy semantics test を struct 形態へ application。
- 期待: original を copy 経由で別変数に bind 後、original の field を `use_mic` / `use_system` で参照しても compile pass + 値が正しい。
- 実装ヒント:
  ```rust
  #[test]
  fn requested_transcription_sources_copy_semantics_allow_use_after_move() {
      let original = RequestedTranscriptionSources {
          use_mic: true,
          use_system: false,
      };
      // Copy 派生があれば、let copied = original で move されず、original も使える
      let copied = original;
      // original を copied と独立に使えることを確認
      assert_eq!(original.use_mic, true, "original.use_mic should still be readable (Copy)");
      assert_eq!(original.use_system, false, "original.use_system should still be readable (Copy)");
      // copied 側も値が独立して使える
      assert_eq!(copied.use_mic, true, "copied.use_mic should match");
      assert_eq!(copied.use_system, false, "copied.use_system should match");
  }
  ```

### T3: `requested_transcription_sources_partial_eq_holds_reflexive_and_differs_for_each_field`
- 目的: `#[derive(PartialEq, Eq)]` で field 単位の独立等値判定が機能する契約を CI 固定。
- 期待: 同 field 値 → 等値、片方の field のみ違う → 不等値、両 field 違う → 不等値、を確認。
- 実装ヒント:
  ```rust
  #[test]
  fn requested_transcription_sources_partial_eq_holds_reflexive_and_differs_for_each_field() {
      let a = RequestedTranscriptionSources {
          use_mic: true,
          use_system: false,
      };
      let same = RequestedTranscriptionSources {
          use_mic: true,
          use_system: false,
      };
      let mic_diff = RequestedTranscriptionSources {
          use_mic: false,  // 違う
          use_system: false,
      };
      let system_diff = RequestedTranscriptionSources {
          use_mic: true,
          use_system: true,  // 違う
      };
      let both_diff = RequestedTranscriptionSources {
          use_mic: false,
          use_system: true,
      };

      // reflexive 等値
      assert_eq!(a, same, "same field values should be equal");
      // 片方の field 違いで不等値
      assert_ne!(a, mic_diff, "differs by use_mic");
      assert_ne!(a, system_diff, "differs by use_system");
      // 両 field 違いで不等値
      assert_ne!(a, both_diff, "differs by both");
  }
  ```
