/// フロントエンドに返すオーディオデバイス情報

### T2 = Clone 独立性 (PartialEq 未派生 = Debug 文字列間接判定) (1 件)
- テスト名 `audio_device_clone_is_deep_and_does_not_mutate_original`
- 内容: PartialEq 派生がないため、cloned 側を mutate した後 original の Debug 文字列で間接的に独立性を確認 (mjc-main-35 Loop 1 AppSettings と同じパターン)
  - original = AudioDevice { name: "Mic-A", id: "id-A" } を作成
  - cloned = original.clone()
  - cloned の name と id を別値 (例: "Mic-B" / "id-B") に変更
  - format!("{:?}", original) で original の Debug 文字列を取得し、以下を assert (5 段以上):
    1. original の Debug 文字列に「変更後の cloned の値」("Mic-B" / "id-B") が **含まれていない**
    2. original の Debug 文字列に「original の元値」("Mic-A" / "id-A") が **含まれている**
    3. field 単位独立に 2 fields × (元値あり + 新値なし) で 4 段以上の細かい assert

### T3 = serde Serialize default snake_case JSON key 名固定 (1 件)
