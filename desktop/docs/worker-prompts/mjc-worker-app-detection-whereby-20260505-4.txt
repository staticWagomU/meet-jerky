# タスク: Whereby (`whereby.com` / `<subdomain>.whereby.com`) ミーティング URL 検知の追加

AGENTS.md 優先順位 2 = 会議検知の網羅性。既存 4 サービス (Google Meet / Zoom / Webex / Teams) に Whereby を追加する。

## 仕様

Whereby のミーティング URL 形式:
- `https://whereby.com/<room-name>` (root 形式、Whereby 公開ルーム)
- `https://<subdomain>.whereby.com/<room-name>` (subdomain 形式、organization 用カスタムサブドメイン、例: `mycompany.whereby.com/team-room`)

通常ページ (約款・ヘルプ等) との区別が必要。Whereby の `whereby.com/` 配下には以下のような non-room path が存在する (blacklist 対象):
- `about`, `pricing`, `blog`, `login`, `signup`, `help`, `terms`, `privacy`, `contact`, `features`, `customers`, `embedded`, `embed`, `information`, `api`, `products`, `integrations`, `security`, `careers`, `status`

## 実装方針 (既存 `is_zoom_meeting_url` / `is_webex_meeting_url` パターン踏襲)

1. **`is_whereby_host(host: &str) -> bool`** を新規追加:
   - `host == "whereby.com"` または `host.ends_with(".whereby.com")` を accept
   - ただし DNS label 偽装 (`fake-whereby.com`, `whereby.com.attacker.example`, `whereby.example.com`) を排除する厳格チェック
   - 既存 `is_zoom_host` (line 699-) や `is_webex_host` (line 787-) と同水準の検証を行う

2. **`is_whereby_meeting_url(host: &str, path: &str) -> bool`** を新規追加:
   - `is_whereby_host(host)` を要件
   - **path が単一 segment 厳格** (= `path.strip_prefix('/')` 後に末尾 `/` を許容しつつ、内部に `/` を含まない、つまり `/<room-name>` のみ accept、`/<room>/extra` は reject)
   - room-name 部分が **空でない** こと
   - room-name が **blacklist (上記 non-room path)** に含まれないこと (lowercase 比較を推奨、ただし既存 `normalize_url_host` で host は lowercase 化されているため room-name 側の case 扱いは既存サービスのパターンに合わせる)
   - blacklist は `const WHEREBY_NON_ROOM_PATHS: &[&str] = &[...];` で定数化

3. **`classify_meeting_url` (line 511-535) に Whereby 分岐を追加**:
   - 既存 `else if is_teams_meeting_url(...)` の **前または後** に `else if is_whereby_meeting_url(&host, &parsed.path) { "Whereby" }` を追加
   - service 文字列は `"Whereby"` (Cap1 + 残小文字、既存 `"Webex"` / `"Zoom"` / `"Microsoft Teams"` 等の命名規則踏襲)

4. **rustdoc の更新は不要** (line 546- は window title fallback 用 rustdoc。今回は URL classify のみ追加なので、line 507-510 の URL classify rustdoc に変更が必要なら最小限の追加を許可するが、必須ではない)
