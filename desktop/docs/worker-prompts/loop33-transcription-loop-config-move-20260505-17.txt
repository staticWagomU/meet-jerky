## C. transcription_worker_loop.rs に struct 定義追加

```rust
pub(crate) struct TranscriptionLoopConfig {
    pub(crate) consumer: ringbuf::HeapCons<f32>,
    pub(crate) source: TranscriptionSource,
    pub(crate) stream: Box<dyn TranscriptionStream>,
    pub(crate) running: Arc<AtomicBool>,

```

- 必要 use 文を transcription_worker_loop.rs の冒頭 use 文ブロックに追加:
  - `use std::sync::atomic::AtomicBool;` (struct field 用、既存 `use std::sync::atomic::Ordering;` の同行 wrap または隣行)
  - `use std::sync::Arc;` (struct field 用)

  - 注: `ringbuf::HeapCons<f32>` / `tauri::AppHandle` / `crate::session_manager::SessionManager` は full path 表記でそのまま使える (use 追加任意、既存スタイル準拠)
