## 想定変更

### 1. 新規ファイル `src-tauri/src/session_manager_types.rs` (~18 行)

```rust
use crate::session::Session;
use chrono::FixedOffset;
use std::path::PathBuf;

/// 活性セッションの状態。
///
/// `output` が `Some` の場合、`append`/`finalize` ごとに対応ファイルへ全文書き出しを行う
/// （インクリメンタル書き出し）。`None` の場合は in-memory のみで動作する。
pub(crate) struct ActiveSession {
    pub(crate) session: Session,
    pub(crate) output: Option<ActiveOutput>,
}

pub(crate) struct ActiveOutput {
    pub(crate) path: PathBuf,
    pub(crate) offset: FixedOffset,
}
```

**重要**:
- visibility は `pub(crate)` (crate 外には漏れない、session_manager.rs から参照可能)
- field も `pub(crate)` (session_manager.rs から `active.session`, `active.output`, `output.path`, `output.offset` 形式でアクセスされる、L97/109/111/136/141/149/165/167/171 で確認済)
- doc comment は session_manager.rs の現定義 (L17-19) からそのまま移動
- use 文は新 file 内に明示 (Session / FixedOffset / PathBuf)
