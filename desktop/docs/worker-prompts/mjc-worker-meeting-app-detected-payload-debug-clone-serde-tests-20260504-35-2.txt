### 追加する test 関数 3 件 (mod tests の末尾に追加)

**T1**: `meeting_app_detected_payload_debug_output_contains_variant_names_and_all_field_names`
- 目的: MeetingAppDetectedPayload の Debug 出力に 2 variants 名 + 全 field 名 + 値が含まれる契約を CI 固定 (logger 経由のデバッグ性保護)。
- 手順:
  - `let app_variant = MeetingAppDetectedPayload::App { bundle_id: "us.zoom.xos".to_string(), app_name: "zoom.us".to_string() };`
  - `let browser_variant = MeetingAppDetectedPayload::Browser { bundle_id: "com.google.Chrome".to_string(), app_name: "Google Chrome".to_string(), service: "Google Meet".to_string(), url_host: "meet.google.com".to_string(), browser_name: "Chrome".to_string() };`
  - `let app_dbg = format!("{app_variant:?}");`
  - `let browser_dbg = format!("{browser_variant:?}");`
  - assert: `app_dbg.contains("App")` (variant 名)
  - assert: `app_dbg.contains("bundle_id")`, `app_dbg.contains("app_name")` (snake_case field 名)
  - assert: `app_dbg.contains("us.zoom.xos")`, `app_dbg.contains("zoom.us")` (値)
  - assert: `browser_dbg.contains("Browser")` (variant 名)
  - assert: `browser_dbg.contains("bundle_id")`, `browser_dbg.contains("app_name")`, `browser_dbg.contains("service")`, `browser_dbg.contains("url_host")`, `browser_dbg.contains("browser_name")` (snake_case 5 field 名)
  - assert: `browser_dbg.contains("com.google.Chrome")`, `browser_dbg.contains("Google Chrome")`, `browser_dbg.contains("Google Meet")`, `browser_dbg.contains("meet.google.com")`, `browser_dbg.contains("Chrome")` (5 値)
  - 計 16 段程度の `assert!` 単位 assert (2 variants 名 + 7 field 名 + 7 値)。

**T2**: `meeting_app_detected_payload_clone_is_deep_and_does_not_mutate_original`
- 目的: MeetingAppDetectedPayload の `Clone` derive が deep clone である契約を CI 固定 (PartialEq 未派生対応として Debug 文字列で間接判定)。
- 手順:
  - `let original = MeetingAppDetectedPayload::Browser { bundle_id: "com.apple.Safari".to_string(), app_name: "Safari".to_string(), service: "Microsoft Teams".to_string(), url_host: "teams.microsoft.com".to_string(), browser_name: "Safari".to_string() };`
  - `let cloned = original.clone();`
  - `let cloned_dbg = format!("{cloned:?}");`
  - cloned 側の値の存在を Debug 文字列で確認 (deep clone されていれば値が同じ Debug 出力に再現される):
    - `assert!(cloned_dbg.contains("Browser"));`
    - `assert!(cloned_dbg.contains("com.apple.Safari"));`
    - `assert!(cloned_dbg.contains("Safari"));`
    - `assert!(cloned_dbg.contains("Microsoft Teams"));`
    - `assert!(cloned_dbg.contains("teams.microsoft.com"));`
  - original 側を新しい値で **再束縛** (let mut + 別 enum 値) し、cloned が影響を受けないことを確認:
    - `let original = MeetingAppDetectedPayload::App { bundle_id: "DIFFERENT".to_string(), app_name: "DIFFERENT_NAME".to_string() };` (re-binding)
    - `let _ = original;` (unused warning 回避)
    - `let cloned_dbg_after = format!("{cloned:?}");`
    - `assert!(cloned_dbg_after.contains("Browser"), "cloned: 再束縛後も Browser variant 維持");`
    - `assert!(cloned_dbg_after.contains("com.apple.Safari"), "cloned: 元の bundle_id 維持");`
    - `assert!(cloned_dbg_after.contains("Microsoft Teams"), "cloned: 元の service 維持");`
    - `assert!(!cloned_dbg_after.contains("DIFFERENT"), "cloned: 再束縛後の値混入なし");`
  - 注意: enum は struct と違って field を直接 mutate できない (`cloned.bundle_id = ...` は variant が match される必要がある)。簡潔さのため `let original = ...` の **再束縛** 手法で「元変数を別 instance へ置き換えても cloned は影響を受けない」契約を Debug で検証する。これは「変数 binding と heap data の分離」を CI 固定する変奏。
  - 計 10 段程度の assert (5 cloned 値存在 + 5 cloned 影響なし)。

**T3**: `meeting_app_detected_payload_serde_serialize_uses_tagged_enum_with_field_level_rename`
- 目的: MeetingAppDetectedPayload の `#[serde(tag = "source")]` + field-level `#[serde(rename = "...")]` JSON contract を CI 固定 (Frontend 側 Tauri event payload 構造との contract 保護、tag 名 "source" や field 名 "bundleId" 等を保護)。
- 手順:
  - **App variant** test:
    - `let app = MeetingAppDetectedPayload::App { bundle_id: "us.zoom.xos".to_string(), app_name: "zoom.us".to_string() };`
    - `let json = serde_json::to_value(&app).expect("serialize ok");`
    - `let obj = json.as_object().expect("object");`
    - assert: `obj.len() == 3` (source tag + 2 fields)
    - assert: `obj.contains_key("source")` (tag key)
    - assert: `obj.contains_key("bundleId")` (field-level rename camelCase)
    - assert: `obj.contains_key("appName")` (field-level rename camelCase)
    - assert: `!obj.contains_key("bundle_id")` (snake_case 不在)
    - assert: `!obj.contains_key("app_name")` (snake_case 不在)
    - assert: `obj["source"] == json!("app")` (tag value, App variant が #[serde(rename = "app")])
    - assert: `obj["bundleId"] == json!("us.zoom.xos")`, `obj["appName"] == json!("zoom.us")` (値)
  - **Browser variant** test:
    - `let browser = MeetingAppDetectedPayload::Browser { bundle_id: "com.google.Chrome".to_string(), app_name: "Google Chrome".to_string(), service: "Google Meet".to_string(), url_host: "meet.google.com".to_string(), browser_name: "Chrome".to_string() };`
    - `let json = serde_json::to_value(&browser).expect("serialize ok");`
    - `let obj = json.as_object().expect("object");`
    - assert: `obj.len() == 6` (source tag + 5 fields)
    - assert: `obj.contains_key("source")`, `obj.contains_key("bundleId")`, `obj.contains_key("appName")`, `obj.contains_key("service")`, `obj.contains_key("urlHost")`, `obj.contains_key("browserName")`
    - assert: `!obj.contains_key("url_host")`, `!obj.contains_key("browser_name")` (snake_case 不在)
    - assert: `obj["source"] == json!("browser")` (tag value)
    - assert: `obj["bundleId"] == json!("com.google.Chrome")`, `obj["appName"] == json!("Google Chrome")`, `obj["service"] == json!("Google Meet")`, `obj["urlHost"] == json!("meet.google.com")`, `obj["browserName"] == json!("Chrome")`
  - 計 24 段程度の assert (3 + 8 + 13 程度)。
