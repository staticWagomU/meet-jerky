## 作業内容 (3 step)

### Step 1: datetime_fmt.rs を編集 = jst() を module level に移動して `pub(crate)` 化

`#[cfg(test)] mod tests { ... fn jst() -> FixedOffset { ... } ... }` の中の `fn jst()` を **削除** し、その代わり `mod tests` の **直前** (= module level) に `#[cfg(test)] pub(crate) fn jst() -> FixedOffset` を追加する。

具体的には L33-L41 範囲を以下のように変更:

old (L33-L41):
```
}

#[cfg(test)]
mod tests {
    use super::*;
    use chrono::FixedOffset;

    fn jst() -> FixedOffset {
        FixedOffset::east_opt(9 * 3600).unwrap()
    }
```

new:
```
}

#[cfg(test)]
pub(crate) fn jst() -> FixedOffset {
    FixedOffset::east_opt(9 * 3600).unwrap()
}

#[cfg(test)]
mod tests {
    use super::*;
    use chrono::FixedOffset;
```

**注意点**: test module 内の `use super::*;` で jst が import される (super = parent module = jst が定義された module level)。よって test module 内の jst() 使用は無変更で動作する。

### Step 2: 4 file の test module 内 jst() を削除 + use 追加

各 file (`session_manager.rs`, `session_commands.rs`, `session_store_render.rs`, `session_store.rs`) で:

- test module 内 (`#[cfg(test)] mod tests { ... }` の中) の `fn jst() -> FixedOffset { ... }` (3 行) を削除
- 同 test module 冒頭 (use super::*; の直後) に `use crate::datetime_fmt::jst;` を追加 (1 行)

各 callsite (`jst()`) は無変更で動作する (use により名前空間に取り込まれる)。

例: `session_manager.rs:153-155`:

old:
```
    fn jst() -> FixedOffset {
        FixedOffset::east_opt(9 * 3600).unwrap()
    }
```

new: (削除のみ、use 追加は test module 冒頭)

各 file の test module 冒頭で `use super::*;` または `use crate::xxx::*;` の直後あたりに `use crate::datetime_fmt::jst;` を追加。
