# 実装内容 (T1/T2/T3 の 3 件)

`src-tauri/src/settings.rs` の `from_legacy_str` シグネチャ (l.41-52) は:

```rust
pub fn from_legacy_str(value: &str) -> Self {
    match value {
        "local" | "whisper" => Self::Whisper,
        "cloud" | "openAIRealtime" | "openai_realtime" => Self::OpenAIRealtime,
        "elevenLabsRealtime" | "elevenlabs_realtime" | "eleven_labs_realtime" => {
            Self::ElevenLabsRealtime
        }
        "appleSpeech" | "apple_speech" => Self::AppleSpeech,
        _ => Self::Whisper,  // ← fail-safe = ローカル Whisper (課金なし)
    }
}
```

既存 test (l.354-) は known 値 (local/cloud/whisper/openAIRealtime/elevenLabsRealtime/elevenlabs_realtime/appleSpeech) + 不明値 ("unknown") をカバー済。本 test は **catch-all 経路** に到達する 3 つの未保護パターンを CI 固定する。

## T1: 空文字 fail-safe
```rust
#[test]
fn from_legacy_str_returns_whisper_for_empty_string_as_fail_safe() {
    assert_eq!(
        TranscriptionEngineType::from_legacy_str(""),
        TranscriptionEngineType::Whisper,
        "空文字は catch-all で Whisper に倒される fail-safe 契約: 設定欠損時のデフォルトをローカル Whisper (課金なし) に固定"
    );
}
```

## T2: 大文字混在 fail-safe (case sensitivity 設計の固定)
```rust
#[test]
fn from_legacy_str_returns_whisper_for_uppercase_known_value_as_fail_safe() {
    assert_eq!(
        TranscriptionEngineType::from_legacy_str("WHISPER"),
        TranscriptionEngineType::Whisper,
        "known 値 \"whisper\" の大文字版 \"WHISPER\" は catch-all で Whisper に倒される: case sensitivity 設計 (lowercase のみ accept) を CI 固定 = 「将来 case-insensitive 化する誤改修」を遮断する装置 (この test では fail-safe で同じ結果になるが、別 known 値の大文字版で結果が変わる誤改修を間接的に検知)"
    );
}
```

## T3: 空白パディング fail-safe (trim 不在の固定)
```rust
#[test]
fn from_legacy_str_returns_whisper_for_known_value_with_whitespace_padding_as_fail_safe() {
    assert_eq!(
        TranscriptionEngineType::from_legacy_str(" whisper "),
        TranscriptionEngineType::Whisper,
        "known 値 \"whisper\" の前後に空白を含む形は catch-all で Whisper に倒される: trim しない設計を CI 固定 = 「将来 trim 処理を追加する誤改修」を遮断する装置 (この test では fail-safe で同じ結果になるが、trim 化により別 known 値が一致する誤改修を間接的に検知)"
    );
}
```
