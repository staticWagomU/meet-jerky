## 4. 追加する test (3 件、いずれも mod tests 末尾、ファイル末尾の `}` の直前に追加、`#[test]` 必須)

### T1: `session_manager_error_debug_output_contains_each_variant_name_per_variant`
- 目的: `Debug` 出力に「型名 `SessionManagerError`」または各 variant 名 (AlreadyActive / NotActive) が含まれる + 両 variant の Debug 出力が異なる契約を CI 固定。
- 期待:
  - `format!("{:?}", SessionManagerError::AlreadyActive)` に `"AlreadyActive"` が含まれる
  - `format!("{:?}", SessionManagerError::NotActive)` に `"NotActive"` が含まれる
  - 両 Debug 出力が異なる (`assert_ne!`)
- 実装ヒント:
  ```rust
  #[test]
  fn session_manager_error_debug_output_contains_each_variant_name_per_variant() {
      let already = format!("{:?}", SessionManagerError::AlreadyActive);
      let not_active = format!("{:?}", SessionManagerError::NotActive);

      assert!(already.contains("AlreadyActive"), "Debug should contain variant name 'AlreadyActive': {}", already);
      assert!(not_active.contains("NotActive"), "Debug should contain variant name 'NotActive': {}", not_active);

      // 両 variant の Debug 出力が異なる
      assert_ne!(already, not_active, "two variants should produce distinct Debug output");
  }
  ```

### T2: `session_manager_error_display_returns_thiserror_message_for_each_variant`
- 目的: `Display` (= thiserror::Error 派生による `#[error("...")]` 属性) で AlreadyActive → "session already active" / NotActive → "no active session" の文言契約を CI 固定。
- 期待:
  - `SessionManagerError::AlreadyActive.to_string() == "session already active"`
  - `SessionManagerError::NotActive.to_string() == "no active session"`
- 実装ヒント:
  ```rust
  #[test]
  fn session_manager_error_display_returns_thiserror_message_for_each_variant() {
      assert_eq!(
          SessionManagerError::AlreadyActive.to_string(),
          "session already active",
          "AlreadyActive Display should match #[error] attribute"
      );
      assert_eq!(
          SessionManagerError::NotActive.to_string(),
          "no active session",
          "NotActive Display should match #[error] attribute"
      );
  }
  ```

### T3: `session_manager_error_partial_eq_holds_reflexive_and_differs_between_variants`
- 目的: `#[derive(PartialEq)]` で同 variant 等値、異 variant 不等値の契約を CI 固定。
- 期待:
  - `SessionManagerError::AlreadyActive == SessionManagerError::AlreadyActive`
  - `SessionManagerError::NotActive == SessionManagerError::NotActive`
  - `SessionManagerError::AlreadyActive != SessionManagerError::NotActive`
- 実装ヒント:
  ```rust
  #[test]
  fn session_manager_error_partial_eq_holds_reflexive_and_differs_between_variants() {
      // reflexive 等値 (両 variant)
      assert_eq!(SessionManagerError::AlreadyActive, SessionManagerError::AlreadyActive);
      assert_eq!(SessionManagerError::NotActive, SessionManagerError::NotActive);
      // 異 variant 不等値
      assert_ne!(SessionManagerError::AlreadyActive, SessionManagerError::NotActive);
      assert_ne!(SessionManagerError::NotActive, SessionManagerError::AlreadyActive);
  }
  ```
