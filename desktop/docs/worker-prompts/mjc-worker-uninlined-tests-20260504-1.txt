# 実装手順
1. 各箇所を読んで、`format!(...)` / `assert!(...)` / `assert_eq!(...)` / `panic!(...)` / `eprintln!(...)` などの format string で外出しされている変数を特定する。
2. 以下のルールで置換する:
   - 単純な変数名 `{}` + 引数 `var` → `{var}` 直接埋め込み (引数削除)
   - `{:?}` + 引数 `var` → `{var:?}` 直接埋め込み (引数削除)
   - `{:#?}` + 引数 `var` → `{var:#?}` 直接埋め込み
   - `&var` のような借用渡しは、変数名だけ抜き出して `{var}` または `{var:?}` を使う (借用は埋め込み形式でも自動で適用される)
3. 文字列リテラル全体の前後・改行・他の引数は **一切変更しない**。
4. **assertion メッセージのコンテンツ・順序・他の format spec (`:.2`, `:>5` など) は変更しない**。
5. cargo clippy --fix を使うのは **避ける** (他の lint に対する変更も巻き込む可能性があるため)。手動で 1 箇所ずつ Edit する。
