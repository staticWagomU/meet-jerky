ユーザー価値:
ライブ字幕側でも、文字起こしが「文字起こし中」「停止中」「開始中/停止中」など UI state に基づく状態として見え、エラー後にユーザーが録音トラック状態と文字起こし状態を切り分けて判断しやすくなる。

- `transcription-error` payload schema 変更なし。

- 依存関係追加なし。
- 実機権限確認や Tauri AppHandle 抽象化なし。
