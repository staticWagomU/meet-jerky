## 追加する 3 件のテスト

### T1: `engine_default_debug_format_contains_type_name_and_empty_model_field`

`OpenAIRealtimeEngine::default()` の Debug 出力に以下が含まれることを **3 段の assert** で確認する。

- 型名 `"OpenAIRealtimeEngine"` を含む
- field 名 `"model"` を含む (snake_case)
- 空文字列を表す `"\"\""` を含む (`Default` derive で String が `""` になる確認、Debug 出力では二重引用符付きで `""` と表示される)

例:
```rust
#[test]
fn engine_default_debug_format_contains_type_name_and_empty_model_field() {
    let engine = OpenAIRealtimeEngine::default();
    let formatted = format!("{engine:?}");
    assert!(formatted.contains("OpenAIRealtimeEngine"), "型名: {formatted}");
    assert!(formatted.contains("model"), "field 名: {formatted}");
    assert!(formatted.contains("\"\""), "空 String を Debug 出力: {formatted}");
}
```

### T2: `engine_new_with_str_slice_debug_format_contains_provided_model_value`

`OpenAIRealtimeEngine::new("gpt-4o-mini-transcribe")` (= **&str slice** から `Into<String>` 経由でコンストラクト) の Debug 出力に以下が含まれることを **3 段の assert** で確認する。

- 型名 `"OpenAIRealtimeEngine"` を含む
- field 名 `"model"` を含む
- 渡したモデル名 `"gpt-4o-mini-transcribe"` を含む (Debug 出力では二重引用符付きで `"gpt-4o-mini-transcribe"` と表示される)

例:
```rust
#[test]
fn engine_new_with_str_slice_debug_format_contains_provided_model_value() {
    let engine = OpenAIRealtimeEngine::new("gpt-4o-mini-transcribe");
    let formatted = format!("{engine:?}");
    assert!(formatted.contains("OpenAIRealtimeEngine"));
    assert!(formatted.contains("model"));
    assert!(formatted.contains("\"gpt-4o-mini-transcribe\""));
}
```

### T3: `engine_new_with_owned_string_debug_format_contains_provided_model_value`

`OpenAIRealtimeEngine::new(String::from("gpt-4o-transcribe"))` (= **owned String** から `Into<String>` 経由でコンストラクト) の Debug 出力に以下が含まれることを **3 段の assert** で確認する。

- 型名 `"OpenAIRealtimeEngine"` を含む
- field 名 `"model"` を含む
- 渡したモデル名 `"gpt-4o-transcribe"` を含む

= `Into<String>` の **&str path と owned String path 双方を CI でカバー**することで、コンストラクタの API 契約を 2 入力タイプで CI 保護する。

例:
```rust
#[test]
fn engine_new_with_owned_string_debug_format_contains_provided_model_value() {
    let engine = OpenAIRealtimeEngine::new(String::from("gpt-4o-transcribe"));
    let formatted = format!("{engine:?}");
    assert!(formatted.contains("OpenAIRealtimeEngine"));
    assert!(formatted.contains("model"));
    assert!(formatted.contains("\"gpt-4o-transcribe\""));
}
```
